## Clase 9

Para crear un proyecto desde cero utilicen este comando.
~~~
npm init -y
~~~

En el caso del ejemplo para poder ejecutarlo, antes de iniciar el servidor de Node deben entrar primero a la carpeta Backend, abrir una terminal o consola y ejecutar el siguiente comando para instalar las dependencias que est√°n en el archivo package.json y generar la carpeta node_modules.

~~~
npm install
~~~

Y para ejecutarlo se debe utilizar el siguiente comando.

~~~
npm start
~~~
